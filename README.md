## 什么是 ChatGPT镜像站？

镜像网站是指将原始网站的内容复制并放置在另一服务器上的网站。这个概念通常应用于提供备用访问途径，为主站点的繁重流量提供缓解。一般来说，镜像网站会更新以保持与原始网站相同的内容，但这个更新的频率可能因镜像站点的设定不同而不同。
